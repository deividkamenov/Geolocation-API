## HTML5 Geolocation API – Seminar Project

Small static web page in Bulgarian that explains and demonstrates the **HTML5 Geolocation API** and working with geo data in web applications.

### Contents

- `referat.html` – main article page (Bulgarian) with explanations, code snippets and references
- `css/` – styles for the page
- `javascript/` – client‑side scripts (if enabled in later versions)
- `62585.pptx` – presentation slides for the seminar
- `62585_invite.png` – invitation / visual asset

### Topics covered

- What Geolocation and the HTML5 Geolocation API are
- How to retrieve the user’s location in the browser (code example with `navigator.geolocation`)
- Pros and cons of using geolocation in web apps
- Comparison with GPS, IP‑based and Wi‑Fi based location
- Privacy and security considerations when processing location data

### How to run

- Open `referat.html` directly in a modern browser.
- Make sure the browser is allowed to access your location if you want to test the Geolocation demo code.

- Explaining and documenting a web API in Bulgarian
